# 3D Shape Stacking｜立體圖形疊高高

一個適合小學數學課堂使用的互動 3D 立體圖形教具。

## 功能

- 加入正方體、長方體、三角柱及球體
- 拖動及堆疊立體圖形
- 使用物理引擎測試圖形是否站穩
- 提供穩陣及易跌示範
- 支援電腦、iPad 及手機操作
- 可轉動、刪除、清除及重新開始

## 使用技術

- Three.js
- Cannon.js
- HTML / CSS / JavaScript

## 開啟方法

直接開啟 `index.html`，或在 GitHub Repository 的 Settings → Pages，把來源設為 `Deploy from a branch`，選擇 `main` / `(root)`。

## 教學重點

學生可透過實際操作觀察：

- 平面較大的圖形通常較容易站穩
- 球體容易滾動，不適合放在最底層
- 上層圖形需要放近支撐面的中央
- 不同擺放方向會影響穩定性
